Install Creality Print (Flatpak) on Chromebook

Enable Linux

chrome://os-settings → Developers → Turn on Linux development environment

Install Flatpak

sudo apt update && sudo apt install flatpak

Add Flathub Repository

flatpak --user remote-add --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo

Restart Linux

Right-click Terminal → Shut down Linux

Download Creality Print

Download CrealityPrint-Linux-flatpak_V7.0.0-Release_x86_64.flatpak

Share Downloads folder with Linux in Files app.

Install Creality Print

cd /mnt/chromeos/MyFiles/Downloads
flatpak install ./CrealityPrint-Linux-flatpak_V7.0.0-Release_x86_64.flatpak

Run Creality Print

flatpak run com.creality.crealityprint
